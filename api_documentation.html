All India Villages API Documentation

REST API documentation for village search, dropdowns, and address autocomplete

Base URL

Local API Base URL:

http://127.0.0.1:5000/api/v1

Authentication

All protected APIs require an API key.

Header:

X-API-Key: ak_demo_123456789

Browser Testing:

?api_key=ak_demo_123456789

Available Endpoints

GET/states

Returns all available states.

http://127.0.0.1:5000/api/v1/states?api_key=ak_demo_123456789

GET/states/{state_code}/districts

Returns districts for a selected state.

http://127.0.0.1:5000/api/v1/states/27/districts?api_key=ak_demo_123456789

GET/districts/{district_code}/subdistricts

Returns sub-districts for a selected district.

http://127.0.0.1:5000/api/v1/districts/497/subdistricts?state_code=27&api_key=ak_demo_123456789

GET/search?q=Manibeli

Search villages by village name.

http://127.0.0.1:5000/api/v1/search?q=Manibeli&api_key=ak_demo_123456789

GET/autocomplete?q=Man

Returns autocomplete suggestions for village names.

http://127.0.0.1:5000/api/v1/autocomplete?q=Man&api_key=ak_demo_123456789

GET/admin/stats

Returns admin dashboard statistics.

http://127.0.0.1:5000/api/v1/admin/stats?api_key=ak_demo_123456789
